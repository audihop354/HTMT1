## Project Description
Write a program (any language you familiar with) to adopt these requirements:
1. Input a signed integer X (8-bit)
2. Output the binary bit pattern of X in:
    Sign-Magnitude form
    1’s complement form
    2’s complement form
3. Output the hexadecimal form of X (2’s complement)
